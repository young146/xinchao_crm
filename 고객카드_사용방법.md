# 📇 고객 상세 카드 - 사용 방법

## ✨ 새로운 기능

이제 각 고객을 **클릭**하면 상세 정보 카드가 나타납니다!

---

## 🎯 사용 방법

### 1단계: 대시보드로 이동
```
앱 실행 → "📊 데이터 보기" 탭 클릭
```

### 2단계: 고객 선택
- 테이블에서 원하는 고객의 **행 전체를 클릭**
- 예: "Awesome Academy" 행을 클릭

### 3단계: 상세 정보 확인
- 고객 카드 모달이 나타납니다
- 모든 정보를 한눈에 확인 가능

### 4단계: 카드 닫기
- **X 버튼** 클릭
- 또는 **배경(어두운 부분)** 클릭
- ESC 키는 아직 미지원 (추가 예정)

---

## 📋 고객 카드에 표시되는 정보

### 📌 기본 정보
- ✅ 업체명
- ✅ 고객 번호
- ✅ 고객 유형 (대형/중형/소형/옐로우페이지)
- ✅ 주소
- ✅ 전화번호 (자동 분리)
- ✅ 모바일 (자동 분리)
- ✅ 페이지 번호
- ✅ 광고 사이즈

### 📄 계약 정보
- ✅ 호당 단가
- ✅ 총 계약 금액 (자동 계산)
- ✅ 시작 호수
- ✅ 종료 호수
- ✅ 계약 기간 (X개 호)
- ✅ 비고 (Remarks)

### 💰 수금 현황
- ✅ 수금률 (프로그레스 바)
  - 100% 이상: 초록색
  - 50~100%: 주황색
  - 50% 미만: 빨간색
- ✅ 총 계약 금액
- ✅ 수금 완료 금액
- ✅ 미수금

### 📊 호수별 광고 현황
- ✅ Vol 550 ~ Vol 574 범위 내 계약 호수 시각화
- ✅ 예: Vol 552, 553, 554, 555, 556, 557
- ✅ 총 몇 개 호에 광고 게재 중인지 표시

### 🎬 액션 버튼 (준비 중)
- 📞 전화 걸기
- ✉️ 이메일 발송
- ✏️ 정보 수정

## 🎨 UI 특징

### 색상 코드
- **대형광고주 (FC)**: 빨간색 (#d32f2f)
- **중형광고주 (1/2)**: 파란색 (#1976d2)
- **소형광고주 (1/4)**: 초록색 (#388e3c)
- **옐로우페이지**: 주황색 (#f57c00)

### 인터랙티브 요소
- 행에 마우스 오버 → 파란색 하이라이트
- 클릭 가능한 행 → 커서가 포인터로 변경
- 모달 배경 → 클릭하면 닫힘
- X 버튼 → 호버 효과

### 반응형 디자인
- 최대 너비: 900px
- 최대 높이: 90vh (화면 높이의 90%)
- 내용이 많으면 스크롤 가능

## 💡 활용 예시

### 1. 미수금 확인
```
1. 테이블에서 미수금(빨간색)이 있는 고객 찾기
2. 클릭해서 상세 카드 열기
3. 수금률과 미수금 금액 확인
4. "📞 전화 걸기" 버튼으로 연락
```

### 2. 계약 기간 확인
```
1. 고객 클릭
2. "📄 계약 정보" 섹션에서 시작/종료 호수 확인
3. "📊 호수별 광고 현황"에서 시각적으로 확인
```

### 3. 빠른 연락
```
1. 고객 클릭
2. "📋 기본 정보"에서 전화번호/모바일 확인
3. 바로 전화 걸기
```

## 🔧 기술 정보

### 컴포넌트
- `CustomerCard.js` - 고객 상세 카드 컴포넌트
- 모달 방식으로 구현 (position: fixed)
- 배경 클릭 또는 X 버튼으로 닫기

### 데이터 파싱
- `parsePhoneNumbers()` - 전화번호 자동 분리
- `parseVolumeRange()` - 호수 범위 추출
- `parsePrice()` - 가격 파싱
- 자동 계산: 총 계약 금액, 수금률, 계약 기간

### 스타일
- 인라인 스타일 사용
- 그라디언트 배경 (헤더)
- 박스 섀도우 효과
- 호버 애니메이션

## ❓ FAQ

### Q: 카드가 안 나타나요
**A:** 테이블의 행 전체를 클릭하세요. 업체명만 클릭하는 게 아니라 행 어디든 클릭하면 됩니다.

### Q: 배경을 클릭해도 안 닫혀요
**A:** 카드 내부를 클릭하면 안 닫힙니다. 어두운 배경 부분이나 X 버튼을 클릭하세요.

### Q: Vol 컬럼이 안 보여요
**A:** 해당 고객의 Remarks에 호수 정보가 없으면 표시되지 않습니다.

### Q: 수금률이 이상해요
**A:** 총 계약 금액 기준으로 계산됩니다. 수금 완료 / 총 계약 금액 × 100

### Q: 전화 걸기 버튼을 눌러도 안 돼요
**A:** 아직 준비 중인 기능입니다. 현재는 전화번호를 확인하고 직접 전화하세요.
