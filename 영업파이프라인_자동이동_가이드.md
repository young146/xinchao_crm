# 📝 자동 단계 이동 시스템 - 완벽 가이드

## 🎯 핵심 개념

**상담 기록을 작성하면 자동으로 단계가 이동합니다!**

```
📝 상담 폼 작성 → 저장 → 자동으로 다음 단계로!
```

---

## 🔄 자동 이동 규칙

### 규칙 1: 첫 상담 → 상담 단계
```
📞 문의 수렴 단계에서
  ↓
📝 상담 기록 1회차 작성
  ↓
🤝 자동으로 "상담" 단계로 이동!
```

### 규칙 2: 두번째 상담 → 진행 단계
```
🤝 상담 단계에서
  ↓
📝 상담 기록 2회차 작성
  ↓
🚀 자동으로 "진행" 단계로 이동!
```

### 규칙 3: 키워드 감지 → 계약 단계
```
🤝 상담 또는 🚀 진행 단계에서
  ↓
📝 상담 내용에 다음 키워드 입력:
   "계약", "OKAY", "OK", "contract"
  ↓
📝 자동으로 "계약" 단계로 이동!
```

### 규칙 4: 입금 키워드 → 수금 단계
```
📝 계약 단계에서
  ↓
📝 상담 내용에 다음 키워드 입력:
   "입금", "수금", "payment"
  ↓
💰 자동으로 "수금" 단계로 이동!
```

### 규칙 5: 완료 키워드 → 완료 단계
```
💰 수금 단계에서
  ↓
📝 상담 내용에 다음 키워드 입력:
   "완료", "complete"
  ↓
✅ 자동으로 "완료" 단계로 이동!
```

## 📋 상담 기록 작성 방법

### 1단계: 상담 기록 버튼 클릭

```
리드 카드에서:
┌─────────────────────────────┐
│ KOTRA                  🟡   │
│ 박소연                       │
│ 📞 010-0000-0000           │
├─────────────────────────────┤
│ [📝 상담 기록] ← 클릭!      │
│            ↑                │
│      여기를 누르세요!        │
└─────────────────────────────┘
```

### 2단계: 상담 폼 작성

```
┌─────────────────────────────────────┐
│ 📝 KOTRA - 상담 기록 1회차          │
├─────────────────────────────────────┤
│ 💡 자동 단계 이동:                  │
│ • 1회차 상담 → 🤝 상담 단계         │
│ • 2회차 상담 → 🚀 진행 단계         │
│ • "계약" 입력 → 📝 계약 단계        │
├─────────────────────────────────────┤
│ 📞 상담 방식:                       │
│ [📞 전화 상담 ▼]                    │
│                                     │
│ 💬 상담 내용: *                     │
│ ┌─────────────────────────────┐   │
│ │ 전화 통화 완료                │   │
│ │ 단가표 발송 요청받음           │   │
│ │ Full 광고에 관심 있음          │   │
│ │ 3개월 계약 검토 중             │   │
│ └─────────────────────────────┘   │
│                                     │
│ ✅ 결정 사항:                       │
│ [📝 계속 진행 ▼]                    │
│                                     │
│ 📌 다음 액션:                       │
│ [견적서 발송               ]        │
│                                     │
│ 📅 다음 연락 예정일:                │
│ [2026-03-20]                        │
├─────────────────────────────────────┤
│ [✅ 저장 (자동 단계 이동)]          │
└─────────────────────────────────────┘
```

### 3단계: 저장 → 자동 이동!

```
"✅ 저장" 클릭
  ↓
✅ 상담 기록 저장됨
  ↓
🎉 자동으로 "상담" 단계로 이동!
  ↓
알림: "📝 첫 상담 기록이 작성되어 
      KOTRA를 🤝 상담 단계로 이동했습니다!"
```

## 🎯 핵심 기능

### 1. 상담 횟수 자동 추적
```
리드 카드에 표시:
[📝 상담 기록 (3회)] ← 현재까지 3회 상담
```

### 2. 자동 단계 이동
```
상담 기록 작성 → 자동 분석 → 적절한 단계로 이동
```

### 3. 키워드 감지
```
감지 키워드:
• 계약: "계약", "OKAY", "OK", "contract"
• 입금: "입금", "수금", "payment"
• 완료: "완료", "complete"
```

### 4. 수동 선택 가능
```
✅ 결정 사항 드롭다운:
- 계속 진행
- 계약 진행 (→ 계약 단계)
- 입금 확인 (→ 수금 단계)
- 완료 (→ 완료 단계)
```

## 💡 Pro Tips

### Tip 1: 키워드를 활용하세요
```
상담 내용에 키워드만 넣으면 자동 이동!

예시:
"고객이 계약 진행 OKAY 했습니다"
→ 자동으로 계약 단계로!

"입금 확인했습니다"
→ 자동으로 수금 단계로!
```

### Tip 2: 결정 사항 직접 선택
```
키워드 없이도 "✅ 결정 사항"에서 선택 가능:
- [계약 진행] 선택 → 계약 단계로
- [입금 확인] 선택 → 수금 단계로
```

### Tip 3: 상담 기록은 누적됨
```
모든 상담 기록이 저장되어
나중에 히스토리 확인 가능!
(향후 업데이트에서 히스토리 뷰 추가 예정)
```

### Tip 4: 수동 이동도 가능
```
자동 이동이 맞지 않으면
기존 버튼으로 수동 이동 가능:
[→ 🚀] [→ 📝] [→ 💰] 등
```

## 🔄 전체 플로우 요약

```
1. 📞 신규 접수
   ↓ (간편 폼: 고객명, 담당자, 연락처)
   
2. 📝 첫 상담 기록 작성
   ↓ (자동) → 🤝 상담 단계
   
3. 📝 두번째 상담 기록 작성
   ↓ (자동) → 🚀 진행 단계
   
4. 📝 상담 내용에 "계약" 또는 "OKAY"
   ↓ (자동) → 📝 계약 단계
   
5. 📝 상담 내용에 "입금" 또는 "수금"
   ↓ (자동) → 💰 수금 단계
   
6. 📝 상담 내용에 "완료"
   ↓ (자동) → ✅ 완료 단계
```

|----------|------------|
| → 📝 계약 | 계약, OKAY, OK, contract |
| → 💰 수금 | 입금, 수금, payment |
| → ✅ 완료 | 완료, complete |

## 🎉 Before vs After

### Before (수동 이동)
```
❌ 상담 후 수동으로 버튼 클릭
❌ 단계 이동 잊어버림
❌ 상담 내용 별도 기록
```

### After (자동 이동)
```
✅ 상담 기록 작성하면 자동 이동
✅ 단계 이동 잊어버릴 일 없음
✅ 상담 내용과 단계 이동 통합
```
