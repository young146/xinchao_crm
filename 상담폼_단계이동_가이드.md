# 📝 상담 폼 기반 자동 단계 이동 - 사용 가이드

## 🎯 핵심: 다음 단계 선택 드롭다운

**상담 폼에서 "다음 단계"를 선택하면 자동으로 이동합니다!**

```
상담 내용 작성 → 다음 단계 선택 → 저장 → 자동 이동!
```

---

## 🎬 실전 시나리오

### 시나리오 1: 신규 접수 → 첫 상담

```
Step 1: 신규 접수
  "➕ 새 상담 기록" → 고객명, 담당자, 연락처 입력
  → 📞 "문의 수렴" 단계에 추가됨

Step 2: 첫 상담 기록
  1. KOTRA 카드에서 [📝 상담 기록] 클릭
  
  2. 폼 작성:
     • 상담 방식: 📞 전화 상담
     • 상담 내용: "단가표 발송 완료, Full 광고 관심"
     • 🎯 다음 단계: [🤝 상담] 선택 ← 여기!
     • 다음 액션: 견적서 발송
  
  3. "✅ 저장 및 단계 이동"
  
  ✨ 결과: 자동으로 🤝 "상담" 단계로 이동!
```

---

### 시나리오 2: 상담 → 진행

```
Step 1: 두번째 상담
  1. KOTRA 카드 (현재 "상담" 단계)
  2. [📝 상담 기록 (1회)] 클릭
  
  3. 폼 작성:
     • 상담 내용: "견적서 검토 완료, 조건 협의 중"
     • 🎯 다음 단계: [🚀 진행] 선택 ← 여기!
     • 다음 액션: 최종 견적서 발송
  
  4. 저장
  
  ✨ 결과: 자동으로 🚀 "진행" 단계로 이동!
```

---

### 시나리오 3: 진행 → 계약

```
Step 1: 계약 확정
  1. KOTRA 카드 (현재 "진행" 단계)
  2. [📝 상담 기록 (2회)] 클릭
  
  3. 폼 작성:
     • 상담 내용: "계약 진행 OKAY! Vol 556~561, $3,000"
     • 🎯 다음 단계: [📝 계약] 선택 ← 여기!
     • 다음 액션: 계약서 발송
  
  4. 저장
  
  ✨ 결과: 자동으로 📝 "계약" 단계로 이동!
```

## 🎯 "다음 단계 선택" 가이드

### 언제 무엇을 선택하나?

```
⏸️ 현재 단계 유지
  → 기록만 남기고 단계는 그대로
  → 예: "추가 검토 필요", "팔로업 대기"

🤝 상담
  → 첫 통화/이메일 완료
  → 고객과 소통 시작

🚀 진행
  → 견적 요청받음
  → 구체적 진행 시작

📝 계약
  → 계약 합의
  → "OKAY", "진행" 확정

💰 수금
  → 계약서 서명 완료
  → 입금 확인 또는 요청

✅ 완료
  → 모든 절차 완료
  → 광고 제작 시작

⏸️ 보류
  → 예산 문제, 타이밍 문제
  → 나중에 재접촉

❌ 취소
  → 계약 불발
  → 경쟁사 선택, 예산 취소
```

## 💡 Pro Tips

### Tip 1: 명확한 선택이 핵심
```
애매하면: [⏸️ 현재 단계 유지]
확실하면: 다음 단계 선택!
```

### Tip 2: 단계 건너뛰기 가능
```
빠른 고객:
📞 문의 → 바로 [📝 계약] 가능!
```

### Tip 3: 보류/취소 쉽게
```
보류 필요: [⏸️ 보류]
취소 확정: [❌ 취소]
```
