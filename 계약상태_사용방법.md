## 🎯 계약 상태 아이콘

### 테이블에 표시되는 아이콘

| 아이콘 | 의미 | 설명 |
|--------|------|------|
| 🚨 | 연체 | 계약 만료 + 미수금 있음 (즉시 조치 필요!) |
| ✅ | 계약 완료 | 계약 만료 + 수금 완료 (정상 종료) |

| ⏳ | 계약 예정 | 아직 시작 안 됨 |
| ❓ | 정보 없음 | 호수 정보 없음 |

## 📊 계약 상태 유형

### 1. 🚨 연체 (OVERDUE)
```
조건: 계약 만료 + 미수금 있음
색상: 빨간색
배경: 연한 빨간색

예시:
- 계약: Vol 550~555 (종료)
- 현재: Vol 560 진행 중
- 수금: $900 / 총 $2,700
- 미수금: $1,800 ← 🚨 즉시 연락 필요!
```

### 2. 💰 선불 (PREPAID)
```
조건: 광고 게재 중 + 선수금 있음
색상: 파란색
배경: 연한 파란색

예시:
- 계약: Vol 550~557 (8개 호)
- 현재: Vol 553 진행 중 (4개 호 게재됨)
- 수금: $2,700 (전액 선불)
- 기대 수금: $1,350 (4개 호분)
- → 2배 선불! 안정적인 고객
```

### 3. ▶️ 광고 게재 중 (ACTIVE)
```
조건: 현재 호수가 계약 범위 내
색상: 초록색
배경: 연한 초록색

예시:
- 계약: Vol 552~557
- 현재: Vol 555 진행 중
- 남은 호수: 3개 호
```

### 4. ✅ 계약 완료 (EXPIRED + 수금 완료)
```
조건: 계약 만료 + 전액 수금
색상: 초록색
배경: 흰색

예시:
- 계약: Vol 550~555 (종료)
- 수금: $2,700 / $2,700 (100%)
- 미수금: $0
- → 재계약 제안 타이밍!
```

### 5. ⏳ 계약 예정 (UPCOMING)
```
조건: 아직 시작 안 됨
색상: 파란색
배경: 연한 파란색

예시:
- 계약: Vol 560~565
- 현재: Vol 553
- 상태: 7개 호 후 시작
```
