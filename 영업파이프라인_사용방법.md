# 💼 영업 파이프라인 관리 - 사용 방법

## 🎯 개요

**영업 파이프라인**은 고객 문의부터 계약 완료까지 전체 영업 프로세스를 시각적으로 관리하는 시스템입니다.

```
문의 수렴 → 상담 → 진행 → 계약 → 수금 → 완료
   📞      🤝     🚀     📝     💰    ✅
```

---

## 📋 영업 단계별 관리

### 1. 📞 문의 수렴
```
상황: 고객이 처음 문의
필요 서류:
  - 단가표
  - 회사소개서
  - 광고 샘플

다음 액션:
  - 단가표 발송
  - 상담 일정 잡기
  - 회사 소개
```

### 2. 🤝 상담
```
상황: 고객과 미팅/통화 진행 중
필요 서류:
  - 상담일지
  - 미팅메모
  - 고객 요구사항

다음 액션:
  - 견적서 준비
  - 제안서 작성
  - 팔로업 전화
```

### 3. 🚀 진행
```
상황: 견적/제안 단계
필요 서류:
  - 견적서
  - 제안서
  - 광고 시안
  - 계약서 초안

다음 액션:
  - 견적서 발송
  - 조건 협의
  - 수정안 제출
```

### 4. 📝 계약
```
상황: 계약 체결 단계
필요 서류:
  - 계약서
  - 사업자등록증
  - 통장사본
  - 입금 확인

다음 액션:
  - 계약서 발송
  - 서명 확인
  - 입금 요청
```

### 5. 💰 수금
```
상황: 대금 수금 단계
필요 서류:
  - 세금계산서
  - 입금 확인서
  - 수금 확인서
  - 영수증

다음 액션:
  - 세금계산서 발행
  - 입금 확인
  - 광고 제작 시작
```

### 6. ✅ 완료
```
상황: 계약 완료 (고객 관리로 이동)
필요 서류:
  - 완료 보고서
  - 광고 결과물

다음 액션:
  - 고객 관리 탭으로 이동
  - 만족도 조사
```

### 특수 상태

#### ⏸️ 보류
```
상황: 일시 보류
사유: 예산 부족, 타이밍 문제 등
다음 액션: 정기 팔로업, 상황 확인
```

#### ❌ 취소
```
상황: 계약 불발
사유: 가격, 경쟁사 선택, 예산 취소 등
다음 액션: 사유 분석, 6개월 후 재접촉
```

## 📝 새 상담 기록 추가

### 방법 1: 수동 입력

```
1. "➕ 새 상담 기록" 버튼 클릭

2. 폼 작성:
   ┌─────────────────────────────┐
   │ 문의일: [2026-03-15]        │
   │ 고객명: [KOTRA]             │
   │ 담당자: [박소연]            │
   │ 직책: [팀장]                │
   │ 연락처: [010-0000-0000]     │
   │ 이메일: [email@example.com] │
   │ 광고 타입: [Full]           │
   │ 사이즈: [FC]                │
   │ 우선순위: [🟡 보통]         │
   │ 비고: [단가표 요청]         │
   └─────────────────────────────┘

3. "✅ 추가" 클릭

4. 자동으로 "문의 수렴" 단계에 추가됨
```

### 방법 2: Google Sheets 자동 불러오기

```
✅ 자동으로 광고접수인덱스에서 불러옴
✅ 단계는 비고 내용을 기반으로 자동 판정
✅ 수동 조정 가능
```

## 🔔 알림 시스템

```

```

- 📅 3일 경과 (정상, 녹색)
- ⚠️ 8일 경과 (주의, 빨강)

상황: KOTRA에서 전화 문의 옴

5. 자동으로 "문의 수렴" 단계에 추가됨

### 시나리오 2: 상담 진행

```

1. 🔔 "KOTRA 팔로업 예정" 알림 확인

2. KOTRA 카드 클릭 → 상세 모달 열림

3. 전화 통화 진행:
   - 단가표 받았는지 확인
   - 관심도 높음

   - "💬 비고" 추가:
     "전화 통화 완료. 견적서 요청받음. Full 광고 3개월 관심"
   
   - 단계 이동:
     [🤝 상담] 버튼 클릭
   
   - 다음 액션:
     [견적서 준비] 클릭

5. 자동으로 "상담" 단계로 이동
```

---

### 시나리오 3: 견적 발송 & 진행

```
2일 후:

1. KOTRA 카드 클릭

   - "필요 서류" 섹션에서

4. 단계 이동:
   - [🚀 진행] 버튼 클릭

5. 비고 추가:
   "견적서 발송 완료 (2026.03.17)"

6. 자동으로 "진행" 단계로 이동
```

---

### 시나리오 4: 계약 체결

```
1주일 후:

1. KOTRA에서 "진행하겠습니다" 연락

2. KOTRA 카드 클릭

3. 단계 이동:
   - [📝 계약] 버튼 클릭

4. 필요 서류 준비:
   - 계약서 작성
   - 통장사본 준비
   - 사업자등록증 요청

5. 계약서 발송 및 서명 확인

6. 비고 추가:
   "계약서 서명 완료 (2026.03.25)
    Vol 556~561 (6개 호)
    $3,000"

7. 다음 단계:
   - [💰 수금] 단계로 이동 대기
```

---

### 시나리오 5: 수금 완료

```
입금 확인 후:

1. KOTRA 카드 클릭

2. 단계 이동:
   - [💰 수금] 버튼 클릭

3. 필요 서류 작업:
   - 세금계산서 발행
   - 입금 확인서 작성
   - 수금 확인서 발급

4. 비고 추가:
   "입금 완료 (2026.03.28)
    $3,000 전액 입금"

5. 최종 완료:
   - [✅ 완료] 버튼 클릭

6. 자동으로:
   - "📊 고객 관리" 탭에 추가됨
   - 영업 파이프라인에서는 "완료" 단계로 이동
```

## 💰 비즈니스 임팩트

### Before (광고접수인덱스만 사용)

```
❌ 수동 관리: Excel/Google Sheets
❌ 단계 추적: 비고란에 텍스트로만
❌ 팔로업: 기억에 의존
❌ 통계: 수동 집계
❌ 서류: 이메일/폴더에 산재
```

### After (영업 파이프라인 사용)

```
✅ 시각적 관리: 칸반 보드
✅ 단계 추적: 드래그 앤 드롭
✅ 팔로업: 자동 알림
✅ 통계: 실시간 집계
✅ 서류: 통합 관리 (향후)
```

### 예상 효과

```
📈 계약 전환율: +40%
   (놓치는 팔로업 방지)

⏱️ 업무 시간: -60%
   (자동화 & 시각화)

💰 매출: +50%
   (더 많은 리드 처리)

📊 예측 정확도: +80%
   (파이프라인 기반)
```

## 🔧 트러블슈팅

### 문제: 리드가 표시 안 됨

```
해결:
1. F5 (새로고침)
2. Google Sheets 접근 권한 확인
3. 브라우저 콘솔 확인 (F12)
```

### 문제: 단계 이동이 안 됨

```
해결:
1. 리드 카드 클릭하여 모달 열기
2. 하단 단계 버튼 클릭
3. 페이지 새로고침
```

### 문제: 검색이 안 됨

```
해결:
1. 검색어 확인 (띄어쓰기 주의)
2. 전체 버튼 클릭 후 재검색
3. 페이지 새로고침
```
