# 🚀 데이터 변환 도구 빠른 참조 가이드

## ⚡ 빠른 시작 (3단계)

```bash
1. npm start
2. "🔄 데이터 변환 도구" 탭 클릭
3. "🔄 데이터 변환 시작" 버튼 클릭 → CSV 다운로드
```

## 📊 자동 변환 항목

| ✅ 자동 처리됨 | 원본 → 목표 |
|--------------|-----------|
| 업체명 | CUSTOMER → 업체명 |
| 주소 | Address → 주소 |
| 전화/모바일 | Tel → 전화번호 + 모바일 (자동 분리) |
| 광고 사이즈 | Size → 광고 사이즈 |
| 고객 유형 | Size 기반 자동 분류 (FC=대형, 1/2=중형, 1/4=소형) |
| 단가 | Price → 단가 |
| 호수 | Remarks 파싱 → 시작/종료 호수 |
| Vol 컬럼 | 호수 범위 기반 자동 채우기 (Vol 550~574) |
| 총 계약 금액 | 단가 × (종료-시작+1) |
| 누적 미수금 | Have collect → 누적 미수금 |
| 비고 | Remarks → 최종 상담 기록 |

## ❌ 수동 입력 필요 항목

- [ ] 업종
- [ ] 대표자
- [ ] 담당자
- [ ] 이메일
- [ ] 카카오톡
- [ ] 온라인 병행 단가
- [ ] 온라인 기간
- [ ] 온라인 총액
- [ ] 시작일 (날짜)
- [ ] 종료일 (날짜)
- [ ] 계산서 발행

## 🔍 데이터 확인 체크리스트

변환 후 반드시 확인하세요:

```
✅ 전화번호가 올바르게 분리되었는가?
✅ 호수 범위가 정확하게 파싱되었는가?
✅ 총 계약 금액이 맞는가?
✅ Vol 컬럼이 계약 범위에만 채워졌는가?
✅ 미수금 계산이 정확한가?
✅ 고객 유형이 올바르게 분류되었는가?
```

## 🎯 고객 유형 자동 분류 규칙

```
FC              → 대형광고주
1/2, 1/2.       → 중형광고주
1/4, 1/4.       → 소형광고주
yellowpage      → 옐로우페이지
fleamarket      → 프리마켓
```

## 💡 전화번호 분리 예시

```
입력: "028 5417 1885; 0966 950 499"
→ 전화번호: 028 5417 1885
→ 모바일: 0966 950 499

입력: "093 888 5891"
→ 전화번호: 093 888 5891
→ 모바일: (빈칸)
```

## 📐 호수 범위 파싱 예시

```
"552~557"       → 시작: 552, 종료: 557
"549~560"       → 시작: 549, 종료: 560
"553!"          → 시작: 553, 종료: 553
"monthly~Jul'26" → 호수 없음 (수동 입력 필요)
```

## 💰 금액 계산 예시

```
단가: $450
호수: 552~557 (총 6호)
→ 총 계약 금액: $450 × 6 = $2,700
→ Vol 552: $450
→ Vol 553: $450
→ Vol 554: $450
→ Vol 555: $450
→ Vol 556: $450
→ Vol 557: $450
→ Vol 558: (빈칸)
```

## 🚨 자주 발생하는 문제

### ❓ "변환 중 오류 발생"
→ Google Sheets가 "공개" 설정인지 확인

### ❓ CSV 파일에서 한글이 깨짐
→ Google Sheets에 임포트 (자동 UTF-8 인식)

### ❓ Vol 컬럼이 비어있음
→ Remarks에 "숫자~숫자" 형식이 있는지 확인

### ❓ 전화번호가 하나의 셀에 합쳐져 있음
→ 세미콜론(;) 또는 슬래시(/)로 구분되어 있는지 확인

## 📁 파일 다운로드 옵션

### CSV 다운로드
- ✅ Google Sheets 임포트용
- ✅ Excel에서 열기
- ✅ 파일명: `고객관리시트_변환_2026-02-16.csv`

### JSON 다운로드
- ✅ 백업용
- ✅ 프로그래밍 용도
- ✅ 디버깅
- ✅ 파일명: `고객관리시트_변환_2026-02-16.json`

## 📋 Google Sheets 임포트 단계

```
1. 고객관리 시트 열기
2. 파일 → 가져오기 → 업로드
3. CSV 파일 선택
4. 구분 기호: 쉼표
5. 위치: 새 시트 만들기 (권장)
6. 가져오기 클릭
```
