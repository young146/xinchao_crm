# 💼 영업 파이프라인 - 빠른 시작 가이드

## 🎯 두 가지 핵심 질문에 대한 답변

### ❓ Q1: 단계 이동은 어떻게 하나요?

#### ✅ 답변: 3가지 방법이 있습니다!

---

### 방법 1: 빠른 단계 이동 버튼 (가장 쉬움! 🌟)

```
┌─────────────────────────────┐
│ KOTRA                  🟡   │
│ 박소연                       │
│ 📞 010-0000-0000           │
│ 💬 단가표 요청              │
├─────────────────────────────┤
│ [→ 🤝 상담으로] [📋 상세] [⏸️] [❌] │
│      ↑ 이 버튼 클릭!        │
└─────────────────────────────┘

클릭 → 즉시 "상담" 단계로 이동!
```

#### 단계별 버튼:
```
📞 문의 수렴 단계
  → [→ 🤝 상담으로] 버튼

🤝 상담 단계
  → [→ 🚀 진행으로] 버튼

🚀 진행 단계
  → [→ 📝 계약으로] 버튼

📝 계약 단계
  → [→ 💰 수금으로] 버튼

💰 수금 단계
  → [→ ✅ 완료] 버튼
```

#### 추가 버튼:
```
[📋 상세] - 상세 정보 보기
[⏸️] - 보류로 이동
[❌] - 취소로 이동
```

---

### 방법 2: 상세 모달에서 이동

```
1. 리드 카드 클릭
2. 하단 "단계 이동" 버튼 클릭
3. 원하는 단계 선택
```

---

### 방법 3: 드래그 앤 드롭 (향후 추가 예정)

```
리드 카드를 다른 단계로 드래그
```

---

## 🆕 Q2: 신규 접수는 어떻게 하나요?

### ✅ 답변: 간편 접수 폼 (필수 3가지만!)

```
1. "➕ 새 상담 기록" 버튼 클릭

2. 간편 폼 작성:
   ┌─────────────────────────────┐
   │ ➕ 신규 고객 접수 (간편 모드) │
   ├─────────────────────────────┤
   │ 📅 문의일: [2026-03-15]     │
   │                             │
   │ 🏢 고객명 *                 │
   │ [KOTRA              ]       │
   │                             │
   │ 👤 담당자 *                 │
   │ [박소연             ]       │
   │                             │
   │ 📞 연락처 *                 │
   │ [010-0000-0000      ]       │
   │                             │
   │ 💬 메모 (선택)              │
   │ [단가표 요청         ]       │
   │                             │
   │ 💡 자동 설정:               │
   │ • 단계: 📞 문의 수렴        │
   │ • 우선순위: 🟡 보통         │
   ├─────────────────────────────┤
   │ [✅ 접수하기] [✕ 취소]      │
   └─────────────────────────────┘

3. "✅ 접수하기" 클릭

4. 자동으로 "📞 문의 수렴" 단계에 추가됨!
```

### 필수 입력 항목 (3가지만!)
```
1. 🏢 고객명 *
2. 👤 담당자 *
3. 📞 연락처 *
```

### 선택 입력 항목
```
4. 📅 문의일 (기본값: 오늘)
5. 💬 메모 (상담 내용 간단히)
```

## 🔄 실전 예시: 전체 플로우

### 시나리오: KOTRA 고객의 전체 여정

#### 1단계: 신규 접수 📞
```
전화 문의 옴
  ↓
"➕ 새 상담 기록" 클릭
  ↓
간편 폼 작성:
  • 고객명: KOTRA
  • 담당자: 박소연
  • 연락처: 010-0000-0000
  • 메모: 단가표 요청
  ↓
"✅ 접수하기" 클릭
  ↓
✅ "문의 수렴" 단계에 추가됨!
```

#### 2단계: 상담 단계로 이동 🤝
```
단가표 발송 완료
  ↓
KOTRA 카드에서
  ↓
[→ 🤝 상담으로] 버튼 클릭
  ↓
✅ "상담" 단계로 이동!
```

#### 3단계: 진행 단계로 이동 🚀
```
전화 상담 완료, 견적서 요청
  ↓
KOTRA 카드에서
  ↓
[→ 🚀 진행으로] 버튼 클릭
  ↓
✅ "진행" 단계로 이동!
```

#### 4단계: 계약 단계로 이동 📝
```
견적 승인, 계약 진행
  ↓
KOTRA 카드에서
  ↓
[→ 📝 계약으로] 버튼 클릭
  ↓
✅ "계약" 단계로 이동!
```

#### 5단계: 수금 단계로 이동 💰
```
계약서 서명 완료
  ↓
KOTRA 카드에서
  ↓
[→ 💰 수금으로] 버튼 클릭
  ↓
✅ "수금" 단계로 이동!
```

#### 6단계: 완료 ✅
```
입금 확인 완료
  ↓
KOTRA 카드에서
  ↓
[→ ✅ 완료] 버튼 클릭
  ↓
✅ "완료" 단계로 이동!
  ↓
이제 "📊 고객 관리" 탭에 추가됨
```

## 💡 팁 & 트릭

### 팁 1: 빠른 접수
```
고객 전화 → 즉시 폼 열기 → 3가지만 입력 → 접수
⏱️ 소요 시간: 30초!
```

### 팁 2: 한 번 클릭으로 이동
```
리드 카드의 [→ 다음 단계] 버튼만 클릭
더 이상 복잡한 모달 필요 없음!
```

### 팁 3: 나중에 추가 정보 입력
```
일단 간편하게 접수 → 나중에 [📋 상세]에서 추가 정보 입력
서두르지 마세요!
```

### 팁 4: 보류/취소 관리
```
애매한 고객?
→ [⏸️] 보류 버튼

확실히 안 됨?
→ [❌] 취소 버튼
```

## 📞 자주 묻는 질문 (FAQ)

### Q: 접수 후 단계를 건너뛸 수 있나요?
```
A: 네! [📋 상세] 버튼 클릭 → 하단 단계 버튼에서 원하는 단계 선택
   예: 문의 수렴 → 바로 계약 단계로 가능
```

### Q: 실수로 잘못된 단계로 옮겼어요!
```
A: 다시 [📋 상세] 클릭 → 올바른 단계로 이동
   또는 해당 단계 칸반에서 카드 찾아 다시 이동
```

### Q: 입력한 정보를 수정하려면?
```
A: [📋 상세] 버튼 클릭 → 상세 정보 확인/수정
   (향후 업데이트에서 직접 수정 기능 추가 예정)
```

### Q: Google Sheets와 동기화되나요?
```
A: 현재는 읽기만 가능 (광고접수인덱스에서 자동 불러옴)
   수동 입력한 데이터는 브라우저에만 저장됨
   (향후 업데이트에서 Google Sheets 쓰기 기능 추가 예정)
```
